# shiny-training-2017-05-genentech

Slides and demo materials for the 2-day Intermediate Shiny Workshop at Genentech, May 2017

## Required packages

- cranlogs
- DT
- flexdashboard
- gapminder
- jsonlite
- lubridate
- miniUI
- shiny
- shinyBS
- shinydashboard
- shinyjs
- shinythemes
- shinytoastr
- tidyverse (specifically using dplyr, ggplot2, stringr)
